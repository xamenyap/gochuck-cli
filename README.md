# gochuck-cli
A CLI program to display Chuck Norris jokes. Required Go version 1.16 and above.

## Usage
To build the binary file, simply run
```
gochuck-cli
```
Run the built binary to get a random Chuck Norris Joke
```
./gochuck-cli
```
